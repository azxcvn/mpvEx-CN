# mpvEx 中文版

此项目基于 [mpvEx](https://github.com/marlboro-advance/mpvEx) 进行中文本地化。

## 主要修改

- 将界面和文案全部翻译为中文
- 移除捐赠相关内容

## 原项目特性

mpvEx 是一款基于 mpv-android 的 Android 媒体播放器，使用 Jetpack Compose 构建：

- Material 3 设计
- 画中画播放
- 后台播放
- 网络流媒体支持（SMB/FTP/WebDAV）
- 外部字幕和音频支持
- 文件管理器
- 完全开源，无广告
